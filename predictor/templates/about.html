About Livestock Prognosis

Livestock Prognosis is an intelligent health assessment system designed to help farmers identify diseases early and improve overall herd welfare. Using a combination of advanced machine learning models and a structured symptom analysis workflow, the platform delivers reliable, real-time insights for livestock health monitoring.

The system integrates three powerful classifiers to analyze symptoms and predict the most probable disease outcomes:

  • CatBoost Classifier for handling complex non-linear patterns
  • Random Forest Classifier for stable and interpretable predictions
  • XGBoost Classifier for high-performance gradient boosting accuracy

Along with predictive modeling, Livestock Prognosis is built using the Django framework to ensure a secure, fast and scalable backend architecture. The platform also features a Retrieval-Augmented Generation module, powered by OpenAI, enabling users to query a knowledge base and receive context-aware explanations and expert-level livestock guidance.

Together, these components create a robust, AI-driven decision support system aimed at empowering veterinarians, farmers, and livestock caretakers with instant disease insights and intelligent recommendations.
